Before you start

neoan3 uses common web technologies in order to facilitate a reliable yet simple setup. We encourage local development ( and mention that in {{currentYear}} as neoan3's structure enables you to directly work on live systems ).

neoan3 is designed for Apache & Nginx but setup is easiest on Apache.
