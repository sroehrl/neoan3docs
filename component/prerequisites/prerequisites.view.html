Before you start

neoan3 uses common web technologies in order to facilitate a reliable yet simple setup. We encourage local development ( and mention that in 2019 as neoan3's structure enables you to directly work on live systems ).

neoan3 is designed for Apache & Nginx but setup is easiest on Apache.

Ensure PATH variable

Make sure all commands work properly and are added to PATH.
composer -V
git --version
node -v
npm -v

The cli tool

While you do not need neoan3-cli, it helps you a lot. After installation of npm, install the tool by running npm i -g neoan3-cli.
