Apps

Expanding functionality of Neoan3

Neaon3 is a very light framework that has a lot of functionality out-of-the-box, however, there is a plethora of apps that are available to add as needed. Neoan3 apps not only extend the functionality of the Neoan3 framework but they can be used outside of the framework as well as an add-in.

Sources of Apps

There are three primary sources for Neoan3 Apps:

1. NPM

2. Composer (via Packagist)

3. Github

Practical Application

Regardless of what source you use, you only have to add what you need. There are two main governing principles to creating a Neoan3 App:

1. It is good practice to have the naming convention be neoan3-{{appName}}, albeit not required

2. Build agnostic from the framework. This is to make it possible to use in other frameworks.
